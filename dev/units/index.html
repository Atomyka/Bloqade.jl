Working with Units

Unit conversion can be tedious when working with simulation of real quantum experiments. In Bloqade, the physical variables can support explicit units from Unitful. When using explicit units, they will be converted to our default unit set automatically. In other words, we can use a different unit set to create the Hamiltonian other than the default units. For detailed information about the detault unit of Hamiltonian, please refer to Bloqade. 

julia> using Bloqade
julia> using Unitful: kHz, µm
julia> rydberg_h([(1, ), (2, )], C=109.2kHz * µm^6)∑ 2π ⋅ 0.174e-1.0/|r_i-r_j|^6 n_i n_j

In the above example, we have assigned the the parameter C with the unit kHz * µm^6, which is automatically converted to default unit MHz * µm^6 in Bloqade.

Other than the Hamiltonian, we can also specify units on waveforms, e.g.
