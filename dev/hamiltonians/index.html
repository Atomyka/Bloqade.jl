Hamiltonians

The simplest way of creating a Hamiltonian is via the rydberg_h function

EaRydCore.rydberg_h — Function
rydberg_h(atoms; [C=2π * 109.133 * MHz*µm^6], Ω[, ϕ, Δ])

Create a rydberg hamiltonian

\[∑ \frac{C}{|r_i - r_j|^6} n_i n_j + \frac{Ω}{2} σ_x - Δ σ_n\]

shorthand for

RydInteract(C, atoms) + XTerm(length(atoms), Ω, ϕ) + ZTerm(length(atoms), Δ)

Arguments

  • atoms: a collection of atom positions

Keyword Arguments

  • C: optional, default unit is MHz*µm^6, interation parameter, see also RydInteract.
  • Ω: required, default unit is MHz, Rabi frequencies, see XTerm.
  • Δ: optional, default unit is MHz, detuning parameter, see NTerm.
  • ϕ: optional, does not have unit, the phase, see XTerm.
Tips

The parameters of Hamiltonian have their own default units to match hardware, one can use Unitful.jl to specify their units explicitly. If the units are specified explicitly, they will be converted to default units automatically.

Example

julia> using EaRyd

julia> atoms = generate_sites(SquareLattice(), 3, 3);

julia> rydberg_h(atoms; Δ=1.2, Ω=1.1)
Hamiltonian
  Term 1
   ∑(n=1:9) 1.1/2 σ^x

  Term 2
   -∑(n=1:9) 1.2 n

  Term 3
   ∑(n=1:9) 686.0/|r_i - r_j|^6 n_i n_j
julia> rydberg_h(atoms; Δ=1.2, Ω=1.1, ϕ=2.1)
Hamiltonian
  Term 1
   ∑(n=1:9) 1.1/2 (e^{2.1i}|0)⟨1| + e^{-2.1i}|1⟩⟨0|)

  Term 2
   -∑(n=1:9) 1.2 n

  Term 3
   ∑(n=1:9) 686.0/|r_i - r_j|^6 n_i n_j

The Hamiltonian Expressions

Inside this package, a more general definition of hamiltonians are supported as Symbolic expressions, this gives user the flexiblity to define various different kind of hamltonian by simply writing down the expression.

There are currently 4 terms supported: RydInteract, XTerm, NTerm, ZTerm. The terms can be added up to compose a new hamiltonian, e.g

julia> using EaRyd
julia> h = XTerm(5, 1.0) + ZTerm(5, 1.0)Hamiltonian
  Term 1
   ∑(n=1:5) 1.0/2 σ^x

  Term 2
   ∑(n=1:5) 1.0 σ^z

Convert Hamiltonian Expression to Matrices

The Hamiltonian expressions can be converted to matrices via type conversion, e.g we can convert the above hamiltonian to a SparseMatrixCSC
