Waveforms

Waveforms are essential ingredients for Rydberg quantum simulations. By controlling the waveforms of $\Omega$ and $\Delta$, one can prepare ground states of certain target Hamiltonian and study non-equalibrium dynamics of time-dependent Hamiltonians. With EaRyd, we support several built-in waveforms and allow the users to specify waveforms by inputing functions. We also support different operations of waveforms, such as smoothing, slicing, and composing, et al.

The generated waveforms can be directly used to build time-dependent Hamiltonians, see Hamiltonian.

Creating Waveforms

In EaRyd, the waveforms are defined as Waveform object, which is a composition of a callable object and a real number duration.

EaRyd gives users the flexibility to specify general waveforms by inputing functions. The following code constracting a sinusoidal waveform with time duration of $4 \pi$

julia> using EaRyd
julia> using EaRydPlots
julia> waveform = Waveform(t->2.2sin(t), duration=4π);
julia> draw(waveform)Figure()

We also support several built-in time-dependent waveforms, including piecewise_linear, piecewise_constant, linear_ramp, constant, sinusoidal. For example, the codes below create different waveform shapes simply by single lines 

julia> waveform = piecewise_linear(clocks=[0.0, 0.2, 0.5, 0.8, 1.0], values=[0.0, 1.5, 3.1, 3.1, 0.0]);
julia> draw(waveform)Figure()
julia> waveform = piecewise_constant(clocks=[0.0, 0.2, 0.5], values=[0.0, 1.5, 3.1]);
julia> draw(waveform)Figure()
julia> waveform = linear_ramp(duration=0.5, start_value=0.0, stop_value=1.0);
julia> draw(waveform)Figure()
julia> waveform =  constant(duration=0.5, value=2.1);
julia> draw(waveform)Figure()
julia> waveform =  sinusoidal(duration=4π, amplitude=2.2);
julia> draw(waveform)Figure()

Operations of Waveforms

EaRyd also supports several operations of the waveforms. Waveforms can be sliced using the duration syntax start..stop, e.g


julia> wf = sinusoidal(duration=2.2);ERROR: UndefVarError: sinusoidal not defined
julia> wf[1.1..1.5];ERROR: UndefVarError: .. not defined
julia> draw(wf)ERROR: UndefVarError: draw not defined

Waveforms can be composed together via append

julia> wf1 = Waveform(sin, duration=2.2)ERROR: UndefVarError: Waveform not defined
julia> wf2 = linear_ramp(;start_value=0.0, stop_value=1.1, duration=0.5)ERROR: UndefVarError: linear_ramp not defined
julia> waveform = append(wf1, wf2)ERROR: UndefVarError: append not defined

where the waveform w2 is appended at the end of w1.

Sharp waveforms may result in bad performance in practice (e.g. for adibatic preparing a ground state of target Hamiltnonian), it is sometimes preferred to smoothen the waveform using the moving average methods, one can use the smooth function to create a smooth-ed wavefrom from a piecewise linear waveform.

julia> wf = piecewise_linear(clocks=[0.0, 2.0, 3.0, 4.0], values=[0.0, 3.0, 1.1, 2.2])ERROR: UndefVarError: piecewise_linear not defined
julia> swf = smooth(wf)ERROR: UndefVarError: smooth not defined

Waveform Linear Algebra

EaRyd also supports several linear algebra operations of waveforms. If two waveforms have the same duration, we can directly add up or subtract the strength of two waveforms, simply by using + or -. 

julia> wf1 = linear_ramp(;duration=2.2, start_value=0.0, stop_value=1.0);ERROR: UndefVarError: linear_ramp not defined
julia> wf2 = Waveform(sin, duration=2.2);ERROR: UndefVarError: Waveform not defined
julia> wf3 = wf1 + wf2ERROR: UndefVarError: wf1 not defined
julia> wf4 = wf1 - wf2ERROR: UndefVarError: wf1 not defined

If we want to increase the strength of a waveform by some times, we can directly use *

julia> wf = linear_ramp(;duration=2.2, start_value=0.0, stop_value=1.0);ERROR: UndefVarError: linear_ramp not defined
julia> wf_t = 3 * wfERROR: UndefVarError: wf not defined

Such operation could also be broadcasted by using .*

julia> wf2, wf3 = [2.0, 3.0] .* wf1ERROR: UndefVarError: wf1 not defined
